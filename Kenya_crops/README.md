# Kenya Crops 101

## Overview

This report provides an introductory analysis of crop production in Kenya. It is designed to give users a foundational understanding of Kenya's agricultural landscape, including key crops, production trends, and regional distribution.

## Purpose

The report serves as a reference point for anyone seeking to understand the basics of crop farming in Kenya. It is useful for agricultural analysts, policymakers, researchers, and students interested in Kenya's farming sector.

## Contents

This Power BI report covers the following areas:

**Crop Overview**
A summary of the major crops grown in Kenya, categorized by type, such as food crops, cash crops, and horticultural produce.

**Regional Distribution**
A breakdown of crop production by county or region, highlighting areas with the highest agricultural activity.

**Production Trends**
Visual representations of crop output over time, showing growth, decline, or stability in production volumes.

**Key Metrics**
Core agricultural indicators include total production volumes, acreage under cultivation, and yield rates per crop.

## Data Source

The data used in this report is sourced from publicly available agricultural records and government datasets related to Kenyan crop production.

## How to Use

Open the file in Microsoft Power BI Desktop to interact with the report. Use the filters and slicers provided on each page to explore data by region, crop type, or time period.

## Requirements

Microsoft Power BI Desktop is required to open and interact with this file. It can be downloaded for free from the official Microsoft website.
